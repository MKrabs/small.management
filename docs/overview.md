# Overview

## Core Idea
A calendar event planning web app to help groups of friends find common time slots. Similar in purpose to When2Meet or Doodle, but privacy-oriented, self-hostable, and optimized for mobile touch.

## Positioning
- Open-source, self-hostable
- Aggressively privacy-oriented — no ad trackers, no invasive data collection
- No account required to participate
- Responsive web app, mobile-first for the core interaction (slot painting)

## Licensing
Fair-code (Functional Source License). Free for private and community use. Paid for commercial use.

## Core Principles
- **Frictionless entry** — guests can submit and edit availability without creating an account or verifying an email
- **No admin** — every member of an activity has equal power; the group governs itself
- **Platform, not enforcer** — the service provides tools, not rules; deadlines and structure are optional, never enforced
- **Privacy by default** — activities are invite-link-only; no public discovery page
